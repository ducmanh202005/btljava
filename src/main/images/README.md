# Thư mục Images

Đặt các file hình ảnh game vào đây:

- `player.png` - Hình ảnh tàu người chơi
- `enemy.png` - Hình ảnh kẻ địch
- `bullet.png` - Hình ảnh đạn bắn
- `explosion.png` - Hình ảnh hiệu ứng nổ

Kích thước đề xuất:

- Player: 40x30 pixels
- Enemy: 30x30 pixels
- Bullet: 4x10 pixels
- Explosion: 30x30 pixels
